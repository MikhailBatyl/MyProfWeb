# MyProfWeb

Одностраничный landing-page компании «8Бит» на чистых HTML, CSS и Vanilla JavaScript.

## Как запустить локально

Откройте терминал в корне проекта:

```powershell
cd a:\MyProfWeb\MyProfWeb
```

### Вариант 1: через Python

```powershell
python -m http.server 8000
```

Затем откройте в браузере:

```text
http://localhost:8000
```

### Вариант 2: напрямую

Можно открыть файл `index.html` в браузере двойным кликом или через контекстное меню `Open with Live Server`, если установлен соответствующий плагин для Cursor/VS Code.

> Для корректного отображения hero-изображения файл `8BIT.jpg` должен лежать рядом с `index.html` в корне проекта.
